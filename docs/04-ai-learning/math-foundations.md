## 🔑 线性代数核心概念

### 向量与矩阵

```python
import numpy as np

# 向量（1D 数组）
v = np.array([1, 2, 3])

# 矩阵（2D 数组）
A = np.array([[1, 2], [3, 4]])

# 矩阵乘法（神经网络的核心运算！）
B = np.array([[5, 6], [7, 8]])
print(A @ B)  # 或 np.dot(A, B)
```

### 必须掌握的线代概念

- **矩阵乘法**：神经网络的前向传播就是一系列矩阵乘法
- **转置**：`A.T`
- **特征值与特征向量**：PCA 降维的基础
- **点积**：计算相似度（推荐系统、注意力机制）

## 🔑 微积分核心：导数与梯度

梯度下降是训练神经网络的核心算法，理解偏导数就够用。

```
损失函数 L = f(w1, w2, ..., wn)
梯度 = [∂L/∂w1, ∂L/∂w2, ..., ∂L/∂wn]  ← 指向损失增大的方向
更新：w = w - learning_rate × 梯度       ← 向损失减小方向走一小步
```
