## 🔑 图像的本质

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread("photo.jpg")
print(img.shape)    # 输出：(480, 640, 3)  → 高x宽x通道(BGR)

# 转为灰度图（单通道）
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
print(gray.shape)   # 输出：(480, 640)

# 图像就是数组，每个像素是 0-255 的整数
print(img[100, 200])  # 输出：[B, G, R] 三个通道的值
```

## 🔑 卷积神经网络（CNN）直觉理解

```
输入图像 → [卷积层] → [池化层] → [全连接层] → 分类结果
            ↑提取边缘    ↑缩小尺寸   ↑综合决策
            ↑提取纹理
            ↑提取形状
```

CNN 的核心思想：用小的卷积核（如 3×3）在图像上滑动，自动学习图像特征。

## 🔑 嵌入式 CV 的核心挑战

| 挑战 | 原因 | 解决方案 |
|------|------|---------|
| 内存限制 | MCU RAM 只有 KB 级别 | 模型量化（INT8）、选用 MobileNet 等轻量模型 |
| 算力限制 | MCU 主频低，无 GPU | TinyML、CMSIS-NN 优化库 |
| 功耗限制 | 嵌入式产品功耗敏感 | 间歇推理、降频运行 |
