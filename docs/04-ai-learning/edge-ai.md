## 💡 边缘 AI 的核心价值

```
传统方案：传感器 → 上传云端 → AI 推理 → 返回结果
           ↑ 依赖网络  ↑ 高延迟   ↑ 隐私风险

边缘 AI：传感器 → 本地 MCU 推理 → 立即响应
           ↑ 无需网络  ↑ 毫秒级   ↑ 数据不出设备
```

**充电桩应用场景**：
- 充电枪状态识别（替代机械检测）
- 电池健康状态预测
- 异常充电行为识别
- 温升预测（预防性保护）

---

## 🎯 目标：GD32 + 神经网络

GD32F4xx（Cortex-M4）具备 DSP 指令集，理论上可运行小型神经网络。

预计可部署的模型规模：
- 参数量：< 100K
- RAM 占用：< 64KB
- 推理时间：< 100ms
