# 机器学习框架概览

> 机器学习的全局观：了解有哪些工具，每个工具适合做什么，再选择深入方向。

## 📌 ML 技术栈全景图

```
机器学习技术栈
│
├── 数据处理
│   ├── NumPy（数值计算）
│   ├── Pandas（表格数据）
│   └── OpenCV（图像数据）
│
├── 传统机器学习
│   └── scikit-learn（分类、回归、聚类）
│
├── 深度学习框架
│   ├── PyTorch（学术/研究主流，推荐入门）
│   └── TensorFlow / Keras（工业部署常见）
│
├── 大语言模型（LLM）
│   ├── Hugging Face Transformers
│   └── LangChain（应用开发）
│
└── 边缘部署（与嵌入式结合！）
    ├── TensorFlow Lite
    ├── ONNX Runtime
    └── TinyML / Edge Impulse
```

## 🔑 机器学习三大任务

| 任务类型 | 说明 | 典型算法 | 充电桩应用场景 |
|---------|------|---------|-------------|
| 分类（Classification） | 判断属于哪个类别 | SVM、决策树、神经网络 | 故障类型识别 |
| 回归（Regression） | 预测连续数值 | 线性回归、随机森林 | 充电时间预测 |
| 聚类（Clustering） | 无监督分组 | K-Means | 用户充电行为分析 |

---

## 🔑 推荐学习顺序

1. **Python + NumPy + Pandas** → 数据处理能力
2. **scikit-learn** → 跑通第一个 ML 模型（鸢尾花分类）
3. **PyTorch** → 搭建第一个神经网络
4. **项目实战** → 用自己熟悉的数据（充电桩数据）训练模型
