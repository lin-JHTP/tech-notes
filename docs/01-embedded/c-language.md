## 🔑 函数指针与回调函数

嵌入式中常用函数指针实现"回调"机制，例如中断服务注册、事件驱动框架。

```c
/* 函数指针类型定义 */
typedef void (*EventCallback)(uint8_t event);

/* 注册回调 */
void register_callback(EventCallback cb);
```

## 🔑 volatile 关键字

在嵌入式开发中，`volatile` 用于告知编译器不要对该变量进行优化，
常见于：中断服务函数修改的全局变量、内存映射寄存器。

```c
/* 内存映射寄存器必须声明为 volatile */
volatile uint32_t *GPIO_OUT = (uint32_t *)0x40010C0C;
```
