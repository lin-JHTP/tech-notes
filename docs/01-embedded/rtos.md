## 🔑 RTOS 核心概念

| 概念 | 说明 |
|------|------|
| 任务（Task） | 独立运行的执行单元，类似线程 |
| 调度器 | 决定哪个任务运行，基于优先级 |
| 时间片 | 同优先级任务轮流使用 CPU 的时间 |
| 信号量 | 任务同步机制 |
| 互斥锁 | 保护共享资源，防止并发访问 |
| 消息队列 | 任务间传递数据 |

## 🔑 FreeRTOS 常用 API

```c
/* 创建任务 */
xTaskCreate(
    vMyTask,          /* 任务函数 */
    "MyTask",         /* 任务名称（调试用） */
    configMINIMAL_STACK_SIZE, /* 栈大小（字） */
    NULL,             /* 传入参数 */
    1,                /* 优先级（数字越大越高） */
    &xMyTaskHandle    /* 任务句柄 */
);

/* 任务延时（主动让出 CPU） */
vTaskDelay(pdMS_TO_TICKS(100));  /* 延时 100ms */
```
