## 🔑 主控芯片（GD32）

| 项目 | 说明 |
|------|------|
| 系列 | GD32F3xx / GD32F4xx（视产品定义） |
| 架构 | ARM Cortex-M4 |
| 主频 | 最高 120MHz（F3xx）/ 168MHz（F4xx） |
| Flash | 256KB～1MB |
| RAM | 32KB～128KB |
| 常用外设 | USART、I2C、SPI、CAN、ADC、Timer |

---

## 🔑 计量芯片

电能计量是充电桩的核心功能，常用专用计量芯片：
- **RN8302B**：双通道高精度电能计量 IC
- **HLW8032**：单相计量，价格低
- **ATM90E26**：Microchip 出品，精度高

计量数据通常通过 SPI 或 UART 传输给主控。
