# 📓 每日笔记

> 记录每日学习收获、踩坑记录和灵感闪现。不追求完美，贵在坚持。

---

## 🗓️ 使用方式

1. 每次学习后，用 3-5 分钟记录当天的收获
2. 格式不限，关键词、代码片段、感悟都可以
3. 每月末整理一次，把有价值的内容迁移到对应知识模块

---

## 📝 笔记模板

```markdown
## YYYY-MM-DD

**今天学了什么**：

**遇到的问题**：

**解决方法**：

**明天计划**：
```
