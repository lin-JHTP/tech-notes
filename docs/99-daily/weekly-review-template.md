## 📋 使用说明

每周五下午或周末花 30-45 分钟完成周报复盘。

复盘不是"写给别人看的报告"，而是**写给未来的自己看的成长记录**。

#### 🧠 本周技术收获（重点）

**1. FreeRTOS 栈溢出排查方法**

学到了 `configCHECK_FOR_STACK_OVERFLOW = 2` 和 `vApplicationStackOverflowHook` 的用法。
关键实践：**从项目开始就要开启栈检查**，而不是等出问题再加。
详见：[FreeRTOS实战笔记](../01-embedded/freertos-practical-notes.md)

**2. 4G NAT 环境下的 WebSocket 长连接问题**

通过抓包发现 NAT 超时是连接断开的根本原因，而不是 OCPP 协议问题。
解决方案：TCP keepalive（比心跳更底层、更可靠）。
详见：[问题排查日志](debug-log-example.md) - 示例2

**3. 嵌入式 C 编码规范整理**

整理并发布了[嵌入式C编程规范](../01-embedded/embedded-c-coding-style.md)，
形成了可在团队中推广的规范文档。
